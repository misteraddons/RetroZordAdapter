## Saturn / Megadrive Mode

Serial ID is RZordSat

More info at [SaturnLib](https://github.com/sonik-br/SaturnLib).

#### Saturn / Megadrive button mapping

| Saturn | HID | PS3     | Megadrive          |
|--------|-----|---------|--------------------|
| D-PAD  | HAT | D-PAD   |                    |
| A      | 1   | X       |                    |
| B      | 2   | O       |                    |
| C      | 5   | R1      |                    |
| X      | 0   | &#9723; |                    |
| Y      | 3   | &#9651; |                    |
| Z      | 4   | L1      |                    |
| L      | 6   | L2      | "Home" on M30 2.4G |
| R      | 7   | R2      | Mode               |
| Start  | 9   | Start   |                    |
